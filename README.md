# Quantumleap Insights — Website

**Live site:** https://quantumleap.llc

## Structure

| File | Purpose |
|---|---|
| `index.html` | Main homepage |
| `fsi-roi.html` | FSI AI ROI Assessment tool |

## Deployment

This repo is connected to Netlify. Every push to `main` auto-deploys to https://quantumleap.llc.

**Clean URLs configured:**
- `quantumleap.llc/fsi-roi` → serves `fsi-roi.html`

## Local Preview

Open `index.html` in any browser. No build step required — pure HTML/CSS/JS.
